A. 通过数学运算可以知道是对的
    考虑边界情况
    x=-2^31
    y=-2^31+1
    通过编写程序测试发现
   x = 0x80000000 y=0x80000001
  -x  = 0x80000000 -y=7fffffff
   所以为0
B.((x+y)<<4)+y-x==17*y+15*x
   (x+y)<<4 = 16*y+16*x
   (x+y)*16 = 16*y+16*x
   溢出双方都是等效的，可以不做考虑
C.~x+~y+1 == ~(x+y)
   两边加上x+y
   ~x+x + ~y+y + 1 = ~(x+y)+(x+y)
   -1 + -1 + 1 = -1 正确
D.一样的
   unsigned int 和 int 只差2^31
  但这种偏差已经被减运算除去了
  x-y 和 ux-uy表示的只是两者间的偏差
E. ((x>>2)<<2)就是将x最后两位置0
   通过补码的表示方式可以得知
  不管首位是什么，0-31位总是相加的
